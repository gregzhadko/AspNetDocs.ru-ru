## <a name="implementation"></a><span data-ttu-id="87107-140">Реализация</span><span class="sxs-lookup"><span data-stu-id="87107-140">Implementation</span></span>

<span data-ttu-id="87107-149">Для каждого клиентского соединения выполняется трассировка хода выполнения клиента при чтении потока сообщений с помощью курсора.</span><span class="sxs-lookup"><span data-stu-id="87107-149">For each client connection, the client's progress in reading the message stream is tracked using a cursor.</span></span> <span data-ttu-id="87107-150">(Курсор представляет позицию в потоке сообщений.)
