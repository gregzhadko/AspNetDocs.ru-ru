## <a name="implementation"></a><span data-ttu-id="7cf62-130">Реализация</span><span class="sxs-lookup"><span data-stu-id="7cf62-130">Implementation</span></span>

<span data-ttu-id="7cf62-139">Для каждого клиентского соединения клиента выполняется в режиме чтения потока сообщений отслеживается с помощью курсора.</span><span class="sxs-lookup"><span data-stu-id="7cf62-139">For each client connection, the client's progress in reading the message stream is tracked using a cursor.</span></span> <span data-ttu-id="7cf62-140">(Курсор представляет позицию в потоке сообщений).
