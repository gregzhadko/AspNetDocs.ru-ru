# <a name="jquery-cycle-294"></a>jQuery Cycle 2.94

Следующие файлы циклов jQuery размещаются в этой сети CDN:

- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.js) &mdash; полную версию подключаемого модуля jQuery Cycle. Включает в себя не более двух десятков особых эффектов и всех параметров.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.all.min.js) &mdash; версию минифицированные файла выше. Используйте эту версию для рабочих приложений, чтобы повысить производительность веб-сайта.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.js) &mdash; эта версия подключаемого модуля jQuery Cycle меньше и содержит только эффект перехода "Выцветание".
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.min.js) &mdash; версию минифицированные файла выше. Используйте эту версию для рабочих приложений, чтобы повысить производительность веб-сайта.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.js) &mdash; еще меньшей версии подключаемого модуля jQuery Cycle, который содержит только переход «выцветание» без всех параметров.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.94/jquery.cycle.lite.min.js) &mdash; версию минифицированные файла выше. Используйте эту версию для рабочих приложений, чтобы повысить производительность веб-сайта.
