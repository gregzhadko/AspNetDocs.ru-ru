![Автоматическое восстановление после сбоя компьютера](introduction/_static/image7.png)

<span data-ttu-id="250d7-209">Все это происходит автоматически.</span><span class="sxs-lookup"><span data-stu-id="250d7-209">All of this takes place automatically.</span></span> <span data-ttu-id="250d7-210">Вам нужно всего лишь создать веб-сайт и развернуть в нем приложение, используя Windows PowerShell, Visual Studio или портал управления Azure.</span><span class="sxs-lookup"><span data-stu-id="250d7-210">All you need to do is create a web site and deploy your application to it, using Windows PowerShell, Visual Studio, or the Azure management portal.</span></span>
