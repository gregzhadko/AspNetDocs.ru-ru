# <a name="how-do-i-save-and-load-view-state-information-for-a-custom-web-server-control"></a>[Инструкции:] Сохранение и загрузка сведений о состоянии представления для пользовательского серверного веб-элемента управления

от [Крис пикселей](https://twitter.com/chrispels)

В этом видео Крис пикселей будет показано, как сохранять и восстанавливать данные в ViewState пользовательского элемента управления. Во-первых, Узнайте, как сохранять простые значения, содержащие примитивные типы .NET, такие как String, Int32, Boolean и т. д. Затем Узнайте, как сохранять сложные типы, такие как класс, который содержит свойства, хранящие данные. Также обсуждается последовательность обработки сведений о состоянии представления в жизненном цикле страницы ASP.NET.
