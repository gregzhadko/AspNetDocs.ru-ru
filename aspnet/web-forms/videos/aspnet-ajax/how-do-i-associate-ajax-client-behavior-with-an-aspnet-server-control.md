# <a name="how-do-i-associate-ajax-client-behavior-with-an-aspnet-server-control"></a>[Инструкции] Связывание клиентского поведения AJAX с помощью элемента управления сервера ASP.NET?

по [Крис Пелз](https://twitter.com/chrispels)

Узнайте, как создать поведение клиента на основе Microsoft AJAX, которые могут быть связаны с серверным элементом управления ASP.NET на страницу ASPX. Поведение клиента расширяет функциональные возможности стандартного элемента управления ASP.NET через ассоциации, а не путем создания нового пользовательского элемента управления. Таким образом поведение могут использоваться повторно для нескольких экземпляров или типы элементов управления ASP.NET, в зависимости от его реализации.
