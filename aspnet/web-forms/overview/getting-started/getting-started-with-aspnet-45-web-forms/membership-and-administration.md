## <a name="running-the-application"></a>Запуск приложения

В корзине можно запустить приложение сейчас, чтобы увидеть, как можно добавить, удаления и обновления элементов. Корзина для покупок всего будет отражать Общая стоимость всех элементов в корзине.

1. В обозревателе решений нажмите клавишу **F5** запустить пример приложения Wingtip Toys.  
   Откроется браузер и показывает *Default.aspx* страницы.
2. Нажмите кнопку **вход** ссылку в верхней части страницы. 

    ![Членство и Администрирование - журнала в ссылке](membership-and-administration/_static/image2.png)

   *Login.aspx* откроется страница.
3. Используйте следующие имя пользователя и пароль:  
   Имя пользователя: canEditUser@wingtiptoys.com  
   Пароль. PA$ $word1 

    ![Членство и администрирование - страница входа](membership-and-administration/_static/image3.png)
4. Нажмите кнопку **вход** кнопки в нижней части страницы.
5. В верхней части следующей страницы, выберите **администратора** ссылку для перехода к *AdminPage.aspx* страницы. 

    ![Членство и администрирование - Admin ссылку](membership-and-administration/_static/image4.png)
6. Чтобы протестировать проверку входных данных, нажмите кнопку **добавить продукт** кнопки без добавления любые сведения о продукте. 

    ![Членство и администрирование - страницы администрирования](membership-and-administration/_static/image5.png)

   Обратите внимание на то, что отображаются сообщения обязательное поле.
7. Добавьте сведения для нового продукта и нажмите кнопку **добавить продукт** кнопки. 

    ![Членство и администрирование: Добавление продукта](membership-and-administration/_static/image6.png)
8. Выберите **продуктов** из меню навигации вверху, чтобы просмотреть новый продукт был добавлен. 

    ![Членство и администрирование - Показать новый продукт](membership-and-administration/_static/image7.png)
9. Нажмите кнопку **администратора** ссылку, чтобы вернуться на страницу администрирования.
10. В **удалить продукт** раздел страницы, выберите новый продукт, добавленный в **DropDownListBox**.
11. Нажмите кнопку **удалить продукт** кнопку, чтобы удалить новый продукт из приложения. 

    ![Членство и администрирование — удаление продукта](membership-and-administration/_static/image8.png)
12. Выберите **продуктов** в меню навигации вверху, чтобы убедиться, что продукт был удален.
13. Нажмите кнопку **выйти** существует режим администрирования.   
    Обратите внимание, что больше не отображается на панели навигации вверху **администратора** пункта меню.
