## <a name="avoiding-cross-site-scripting-xss-attacks"></a><span data-ttu-id="fb9fc-152">Предотвращение атак с использованием межсайтовых сценариев (XSS)</span><span class="sxs-lookup"><span data-stu-id="fb9fc-152">Avoiding Cross-Site Scripting (XSS) Attacks</span></span>

<span data-ttu-id="fb9fc-153">Когда вы принимаете входные данные от пользователя и повторно отображаете эти данные на веб-сайте, вы потенциально открываете веб-сайт для атак с использованием межсайтовых сценариев (XSS).</span><span class="sxs-lookup"><span data-stu-id="fb9fc-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="fb9fc-154">Теоретически вредоносный злоумышленник может отправить код JavaScript, который выполняется при повторном отображении входных данных.</span><span class="sxs-lookup"><span data-stu-id="fb9fc-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="fb9fc-155">JavaScript можно использовать для кражи паролей пользователей или других конфиденциальных данных.</span><span class="sxs-lookup"><span data-stu-id="fb9fc-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>

<span data-ttu-id="fb9fc-156">Как правило, атаки XSS можно отменять с помощью кодирования HTML любых входных данных, получаемых от пользователя, перед его отображением на веб-странице.</span><span class="sxs-lookup"><span data-stu-id="fb9fc-156">Normally, you can defeat XSS attacks by HTML encoding whatever input you retrieve from a user before displaying it in a web page.</span></span> <span data-ttu-id="fb9fc-157">Однако в HTML-кодировке выходные данные редактора HTML не только закодировать &lt;скрипт&gt; теги, но также будут кодировать все HTML-теги.</span><span class="sxs-lookup"><span data-stu-id="fb9fc-157">However, HTML encoding the output of the HTML Editor would not only encode &lt;script&gt; tags, it would also encode all HTML tags.</span></span> <span data-ttu-id="fb9fc-158">Иными словами, вы потеряли все форматирование, такие как тип шрифта, размер шрифта и цвет фона.</span><span class="sxs-lookup"><span data-stu-id="fb9fc-158">In other words, you would lose all of the formatting such as the font type, font size, and background color.</span></span>
