## <a name="including-a-static-script-file-structure"></a><span data-ttu-id="22ece-183">*Включение структуры статического файла скрипта*</span><span class="sxs-lookup"><span data-stu-id="22ece-183">*Including a Static Script File Structure*</span></span>
