### <a name="asynchronously-reading-and-writing-http-requests-and-responses"></a><span data-ttu-id="79fb9-177">Асинхронное чтение и запись HTTP-запросов и ответов</span><span class="sxs-lookup"><span data-stu-id="79fb9-177">Asynchronously Reading and Writing HTTP Requests and Responses</span></span>

<span data-ttu-id="79fb9-178">В ASP.NET 4 появилась возможность чтения сущности HTTP-запроса в виде потока с помощью метода *HttpRequest. жетбуфферлессинпутстреам* .</span><span class="sxs-lookup"><span data-stu-id="79fb9-178">ASP.NET 4 introduced the ability to read an HTTP request entity as a stream using the *HttpRequest.GetBufferlessInputStream* method.</span></span> <span data-ttu-id="79fb9-179">Этот метод предоставил потоковый доступ к сущности запроса.</span><span class="sxs-lookup"><span data-stu-id="79fb9-179">This method provided streaming access to the request entity.</span></span> <span data-ttu-id="79fb9-180">Однако он выполнялся синхронно, что привязывает поток к длительности запроса.</span><span class="sxs-lookup"><span data-stu-id="79fb9-180">However, it executed synchronously, which tied up a thread for the duration of a request.</span></span>
