# <a name="aspnet-mvc-4-helpers-forms-and-validation"></a>Вспомогательные приложения, формы и проверка в ASP.NET MVC 4

по [Web Слышатся Team](https://twitter.com/webcamps)

[Скачайте комплект учебных материалов по лагеря Web](https://aka.ms/webcamps-training-kit)

В **ASP.NET MVC 4 модели и доступ к данным** в лаборатории вы были Загрузка и отображение данных из базы данных. В этой лаборатории вы добавите **Music Store** приложение возможность изменения этих данных.

С этой целью в сначала создается контроллер, который будет поддерживать действия создания, чтения, обновления и удаления (CRUD) альбомов. Вы создадите шаблон представления индекса преимуществами компонент формирования шаблонов ASP.NET MVC для отображения свойств альбомов в HTML-таблицы. Чтобы улучшить представление, вы добавите пользовательский вспомогательный метод HTML, который будет выполнять усечение длинные описания.

После этого вы добавите, изменить и создать представления, вы можете изменить альбомов в базе данных, с помощью формы элементы, такие как раскрывающиеся списки.

Наконец дадут возможность создавать удаление альбома, а также можно будет предотвратить ввод неверных данных путем проверки входных данных.

Это Практическое лабораторное занятие предполагается, что у вас есть базовые знания о **ASP.NET MVC**. Если вы не использовали **ASP.NET MVC** раньше, мы рекомендуем к превышению **основы ASP.NET MVC** Практическое лабораторное занятие.

Это лабораторное занятие поможет усовершенствования и новые возможности, описанные ранее, применяя незначительные изменения в образец веб-приложение, в папке с исходным кодом.

> [!NOTE]
> Все примеры кода и фрагменты кода включены в Web лагеря комплект обучающих материалов, доступных в [выпуски Microsoft Web/WebCampTrainingKit](https://aka.ms/webcamps-training-kit). Проект, относящиеся к этой лаборатории доступен в [вспомогательные методы ASP.NET MVC 4, формы и проверка](https://github.com/Microsoft-Web/HOL-MVC4HelpersFormsAndValidation).

- Добавление пользовательского вспомогательного метода HTML
- Создание и настройка изменить представление
